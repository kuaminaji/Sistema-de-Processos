🧪 Teste de Correção do Login

⚠️ Importante: Erro SSL_PROTOCOL_ERROR

Se você ver erro "ERR_SSL_PROTOCOL_ERROR":

O browser está forçando HTTPS quando o servidor está em HTTP.

Solução:

  • Chrome: Vá para chrome://net-internals/#hsts e delete domain localhost
  • OU use modo incógnito (Ctrl+Shift+N)
  • OU acesse via http://127.0.0.1:3000 ao invés de localhost

Ver SOLUCAO_SSL_ERROR.md para detalhes completos.
